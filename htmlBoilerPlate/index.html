Vores destination

Mester din egen surdej

Vi afholder både kurser og praktik forløb, som jeg kalder det. Hvor man som begynder eller måske allerede god surdejsamatør kan komme ned og lære alt hvad det indebære at bage godt surdejsbrød - og ikke mindst gøre det på en måde som alle kan finde tid til, selvom deet kan virke uoverskueligt.
